# FAA / Iowa Data

Real-data experiments use Iowa boundary geometry together with FAA-derived obstacle / no-fly-zone geometries.

Typical inputs include:
- `IOWA-BOUNDARY.geojson`
- `FAA-IOWA.geojson`

These files are used by the obstacle-aware partitioning pipeline after spatial preprocessing and reprojection.

If raw files are not distributed directly in this repository, place the processed inputs in this folder.
